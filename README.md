## Installation

Add this line to your application's Gemfile:

```sh
gem "aws-rsync"
```

And then execute:

```sh
$ bundle
```

Or install it yourself as:

```sh
$ gem install aws-rsync
```
## Usage

```sh
aws-rsync hello yourname
```
